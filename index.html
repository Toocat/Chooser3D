Device support

Requires WebGL ready device, falls back to software renderer (currently without antialiasing).

Ease of use

var chooser3D = new THREE.Chooser3D('chooser3DCanvas')
        .addIconObject('images/social/badoo.png')
        .addIconObject('images/social/flickr.png')
        .addIconObject('images/social/plurk.png')
        .addIconObject('images/social/rss.png')
        .addIconObject('images/social/skype.png')
        .addIconObject('images/social/forrst.png')
        .setEventHandler(function (object) {
                console.log(object.getIdent());
        })
        .paint();

API

.slow() || .normal() || .fast() || .veryFast() - determine the slide speed
.setLight(color, intensity) - sets the directional light
.setArrowColor(color) - set the color of the arrows
.setLeftSelector() && setRightSelector() - replace default arrows
.softwareRendering() - forces usage of the software renderer
.addIconObject(imgUrl, ident, width, height) - adds an icon
.addJSONObject(jsonModelUrl, ident, scale, rotation) - adds a JSON model
.setEventHandler(function(object){}) - set the event handler, object is the newly focused object
